## 1. Цель и задача работы

**Цель:** Освоить методы веб-скрапинга данных из веб-страниц.

**Задача:** Спарсить новости по заданным запросам, извлечь структурированные данные и сохранить их в CSV-файлы.

---

## 2. Используемые методы и инструменты

| Компонент | Инструмент/Библиотека |
|-----------|----------------------|
| Запросы к сайту | `requests` |
| Парсинг HTML | `BeautifulSoup` (bs4) |
| Работа с регулярными выражениями | `re` |
| Сохранение данных | `csv.DictWriter` |

**Запросы для поиска новостей:**
```
queries = [
    'нейротехнологии',
    'нейротехнологии и программирование', 
    'ИИ',
    'Машинное обучение',
    'Искусственный интеллект',
    'Нейросети'
]
```

## 3. Структура извлекаемых данных

#### Базовый файл: `scraped_news_data.csv`
| Поле | Описание |

| Идентификатор новости | Целое число из URL |

| Поле | Описание |
|------|----------|
| Идентификатор | Целое число из URL |

## 4. Ключевые особенности реализации

#### Упрощённые селекторы (на основе анализа структуры сайта):
```
# Дата: из атрибута datetime тега <time>
date_element = soup_news.find('time', datetime=True)
news_date = date_element.get('datetime').split('T')[0]

# Просмотры: точный класс
views_el = soup_news.find('span', class_='icon eye')

# Текст: основной блок + очистка от подписей к фото
content_block = soup_news.find('div', class_='content')
for figcaption in content_block.find_all('figcaption'):
    figcaption.decompose()  # удаляем подписи

# Теги: список в ul.tags
tags_block = soup_news.find('ul', class_='tags')
```

#### Обработка вариативности вёрстки:
- Текст собирается из тегов `['p', 'li', 'blockquote', 'h2', 'h3', 'h4']`
- Лишние пробелы и переносы удаляются через `re.sub(r'\s+', ' ', text)`
