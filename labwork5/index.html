Labwork 5 Tutorial

1. Выполнение заданий

Основные шаги:

  1. Проверка данных: Использован метод info() для анализа обучающей и тестовой выборок. Пропуски отсутствуют.
  2. Разделение данных: Целевая переменная Целевая.Цена отделена от признаков с помощью drop().
  3. Обучение моделей:
    • Линейная регрессия (LinearRegression)
    • Случайный лес (RandomForestRegressor)
  4. Валидация: Модели протестированы на тестовой выборке. Рассчитаны метрики MAE и RMSE.

Базовые результаты моделей:

Модель MAE RMSE
Линейная регрессия 126 852.51 201 883.24
Случайный лес (базовая) 81 983.10 148 867.68

2. Анализ важности признаков

С помощью атрибута feature_importances_ были выявлены наиболее и наименее значимые признаки у Random Forest.

Топ-5 наиболее важных признаков:

  1. Оценка риелтора (0.228159)
  2. Жилая площадь (0.202270)
  3. Широта (0.143071)
  4. Ванные (0.086640)
  5. Долгота (0.057589)

Топ-5 наименее важных признаков:

  1. Количество этажей (0.010213)
  2. Состояние (0.012737)
  3. Год реновации (0.012984)
  4. Спальни (0.013177)
  5. Вид на воду (0.020096)

3. Самостоятельная работа

3.1. Исключение незначащих признаков

Из выборки были исключены признаки с наименьшей важностью: Год реновации, Состояние, Количество этажей, Спальни.

Результат:

  • До оптимизации: MAE: 81 983.10, RMSE: 148 867.68

  • После удаления признаков: MAE: 71 431.87, RMSE: 135 530.65

  • Вывод: Качество модели улучшилось, ошибка снизилась примерно на 13%.

3.2. Исследование параметров Random Forest

Была проведена настройка гиперпараметров модели (увеличение количества деревьев, ограничение глубины).

Выбрана оптимальная конфигурация с малым количеством параметров:

RandomForestRegressor(n_estimators=100, random_state=42)

3.3. Исследование других моделей (XGBoost, LightGBM)

Для повышения точности были протестированы алгоритмы градиентного бустинга.

Результаты:

Модель MAE RMSE Примечание
Random Forest (оптимизированный) 71 431.87 135 530.65 Улучшенные резульататов модели по сравнению с изначальной
XGBoost 66 757.84 126 773.25 Наилучший MAE
LightGBM 68 107.83 122 063.76 Наилучший RMSE

Вывод: Модели градиентного бустинга показали результат лучше, чем случайный лес. XGBoost дал наименьшую среднюю абсолютную ошибку, а LightGBM — наименьшую среднеквадратичную ошибку.

4. Заключение

  1. Модель случайного леса работает значительно лучше линейной регрессии на данных задачах.
  2. Удаление неинформативных признаков положительно сказывается на качестве модели и скорости обучения.
  3. Алгоритмы бустинга (XGBoost, LightGBM) являются наиболее точными для данной задачи предсказания цены недвижимости.
  4. Наиболее влияющими факторами цены являются жилая площадь, оценка риелтора и географическое положение (широта/долгота).
