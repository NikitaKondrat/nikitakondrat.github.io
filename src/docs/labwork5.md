#### 3.3. Исследование других моделей (XGBoost, LightGBM)
Для повышения точности были протестированы алгоритмы градиентного бустинга.

**Результаты:**

| Модель | MAE | RMSE | Примечание                                                |
| :--- | :--- | :--- |:----------------------------------------------------------|
